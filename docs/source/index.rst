Welcome to PyBeamDimap's documentation!
=======================================
PyBeamDimap is a Python module that allows users to easily navigate and extract metadata attributes from the BEAM-DIMAP
files used by `SNAP software`_ developed by the European Space Agency.

.. _SNAP software: https://step.esa.int/main/toolboxes/snap/
