reader.core
===========
``core.py`` contains all classes and methods related to data that is found in all BEAM-DIMAP files. This is usually the
data that is found at the top of the BEAM-DIMAP file.

This class is not designed to be directly used by the user. It is designed for the classes in ``PyBeamDimap.missions``.
