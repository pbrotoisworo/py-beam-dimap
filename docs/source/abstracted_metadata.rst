reader.abstracted_metadata
==========================
The ``abstracted_metadata`` class handles all workflows related to parsing the Abstracted Metadata section that is found
in Sentinel-1 metadata.

This class is not designed to be directly used by the user. It is designed for the classes in ``PyBeamDimap.missions``.
